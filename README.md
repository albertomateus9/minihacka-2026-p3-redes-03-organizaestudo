# OrganizaEstudo

Site estático do projeto **OrganizaEstudo**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado). Esta versão funciona como um exemplo genérico e temático do produto, com dados fictícios e uma prévia interativa.

## Ideia

Uma plataforma que cria uma rotina de estudos a partir das dificuldades, do tempo disponível por dia e das datas de provas ou trabalhos.

## Exemplos temáticos

- **Matéria com dificuldade:** O aluno marca Redes como difícil e recebe mais blocos de revisão.
- **Tempo disponível:** Com 1h30 por dia, o site divide estudo, pausa e retomada.
- **Lembrete de prova:** Datas de prova geram alertas e revisões próximas ao prazo.
