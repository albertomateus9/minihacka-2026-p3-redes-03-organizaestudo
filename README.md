# OrganizaEstudo

Site estático do projeto **OrganizaEstudo**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado).

## Ideia

Uma plataforma que cria uma rotina de estudos a partir das dificuldades, do tempo disponível por dia e das datas de provas ou trabalhos.
